Meta Ads Configuration

Enter your Meta Marketing API credentials. These are stored in-memory only (not persisted). You can also set them as environment variables: META_ACCESS_TOKEN, META_AD_ACCOUNT_ID, META_PAGE_ID.

Short-lived tokens expire in ~1 hour. Click "Extend Token" to exchange for a ~60-day token.

Brand Ad Account IDs

Optionally set a different ad account ID per brand. Leave empty to use the default from above.

Claude API

The Claude API key must be set as an environment variable: ANTHROPIC_API_KEY

How to Get Meta Credentials

  1. Go to Meta Business Suite → Business Settings
  2. Create a Meta App at developers.facebook.com (App ID: 910390974704285)
  3. Add the Marketing API product to your app
  4. Generate a User Access Token with ads_management permission
  5. Find your Ad Account ID in Business Settings → Ad Accounts (format: act_XXXXXXXXXX)
  6. Select your Brand & Page from the selector bar above
